<feature>
  <name>KnowledgeStore — Encrypted knowledge note CRUD</name>
  <files>src/knowledge/store.ts, test/knowledge/store.test.ts</files>
  <behavior>
KnowledgeStore is a synchronous class that encrypts/decrypts markdown notes to/from .enc files in the vault's knowledge/ directory.

**Constructor:** `KnowledgeStore(vaultPath, getActiveKey, getKeyById, pipeline?)`
- `vaultPath: string` — absolute path to vault root
- `getActiveKey: () => { keyId: string; key: Buffer }` — returns active encryption key (loose coupling to KeyRing)
- `getKeyById: (keyId: string) => Buffer` — returns decrypted key by ID (supports rotated keys)
- `pipeline?: VaultAuditPipeline` — optional audit pipeline for emitting knowledge events

**Methods:**

1. `writeNote(relativePath: string, content: string): void`
   - Validates path (no traversal outside knowledge/)
   - Calls `getActiveKey()` to get current key + keyId
   - Encrypts `Buffer.from(content, 'utf-8')` via `encrypt(plaintext, key, keyId)` → EncryptedPayload
   - Ensures parent directory exists (`mkdirSync(dirname(filePath), { recursive: true })`)
   - Atomic write: `writeFileSync(tmpPath, JSON.stringify(payload))` then `renameSync(tmpPath, filePath)`
   - Emits `knowledge_note_created` if note didn't exist before, `knowledge_note_updated` if it did
   - File path: `join(vaultPath, 'knowledge', relativePath + '.enc')`

   Cases:
   - writeNote('conditions/diabetes', '# Diabetes\n...') → creates conditions/diabetes.enc
   - writeNote('conditions/diabetes', 'updated content') → overwrites, emits updated event
   - writeNote('../../keys/keyring', 'evil') → throws PathTraversalError
   - writeNote('../../../etc/passwd', 'evil') → throws PathTraversalError

2. `readNote(relativePath: string): string`
   - Validates path (no traversal)
   - Reads file at `join(vaultPath, 'knowledge', relativePath + '.enc')`
   - If file doesn't exist → throws NoteNotFoundError
   - Parses JSON, validates against EncryptedPayloadSchema with Value.Check()
   - If validation fails → throws NoteCorruptedError
   - Extracts `key_id` from payload, calls `getKeyById(payload.key_id)` to get correct key
   - Decrypts via `decrypt(payload, key)`, returns `.toString('utf-8')`
   - If decryption fails (CryptoError) → catches and throws NoteCorruptedError
   - Emits `knowledge_note_read` audit event

   Cases:
   - readNote('conditions/diabetes') after writeNote → returns original content
   - readNote('nonexistent') → throws NoteNotFoundError
   - readNote with corrupted .enc file → throws NoteCorruptedError
   - readNote('../../keys/keyring') → throws PathTraversalError

3. `listNotes(folder?: string): string[]`
   - If folder provided: lists `join(vaultPath, 'knowledge', folder)`, validates no traversal
   - If no folder: lists `join(vaultPath, 'knowledge')`
   - Recursively walks directory tree
   - Returns .enc files as relative paths (relative to knowledge/), without .enc extension
   - Returns empty array if directory doesn't exist or is empty
   - Results sorted alphabetically
   - No audit event emitted for listing (per research recommendation)

   Cases:
   - listNotes() with two notes in conditions/ → ['conditions/diabetes', 'conditions/hypertension']
   - listNotes('conditions') → ['diabetes', 'hypertension'] (relative to conditions/)
   - listNotes('empty-folder') → []
   - listNotes() when no notes exist → []

4. `noteExists(relativePath: string): boolean`
   - Validates path (no traversal)
   - Returns `existsSync(join(vaultPath, 'knowledge', relativePath + '.enc'))`
   - No audit event

   Cases:
   - noteExists('conditions/diabetes') after writeNote → true
   - noteExists('nonexistent') → false
   - noteExists('../../keys/keyring') → throws PathTraversalError

**Path validation (private method):**
- Resolve `join(vaultPath, 'knowledge', relativePath + '.enc')` using `resolve()`
- Check that the resolved path starts with `resolve(join(vaultPath, 'knowledge'))`
- If not → throw PathTraversalError(relativePath)
