1. **LedgerEntryTypeSchema** -- TypeBox Union of 26 string literals:
   - Clinical (10): clinical_encounter, clinical_medication, clinical_allergy, clinical_diagnosis, clinical_problem_list, clinical_lab_result, clinical_imaging_result, clinical_pathology, clinical_procedure, clinical_amendment
   - Care network (3): care_relationship_established, care_relationship_terminated, care_relationship_suspended
   - Access control (4): access_grant_created, access_grant_modified, access_grant_revoked, access_grant_expired
   - Emergency (3): emergency_config_set, emergency_access_triggered, emergency_access_ended
   - Patient-authored (3): patient_note, patient_directive, patient_preference
   - System (3): vault_initialized, key_rotation, sync_record
   Export `LedgerEntryType = Static<typeof LedgerEntryTypeSchema>`

2. **EntryAuthorSchema** -- TypeBox Object with fields:
   - type: Union of 'patient_agent', 'provider_agent', 'system'
   - id: String
   - display_name: String
   - public_key: String (description: 'Base64-encoded DER (SPKI) Ed25519 public key')
   Export `EntryAuthor = Static<typeof EntryAuthorSchema>`

3. **EntryMetadataSchema** -- TypeBox Object with fields:
   - schema_version: Literal('1')
   - entry_type: LedgerEntryTypeSchema
   - author_type: Union of 'patient_agent', 'provider_agent', 'system'
   - author_id: String
   - amends: Optional(String) -- UUID of entry being amended
   - synced_entry: Optional(String) -- UUID of synced entry
   - payload_size: Number -- byte size of plaintext payload
   Export `EntryMetadata = Static<typeof EntryMetadataSchema>`

4. **SignableContentSchema** -- TypeBox Object with fields:
   - id: String
   - timestamp: String
   - entry_type: LedgerEntryTypeSchema
   - author: EntryAuthorSchema
   - payload: String -- plaintext payload as JSON string
   - metadata: EntryMetadataSchema
   Export `SignableContent = Static<typeof SignableContentSchema>`

5. **LedgerEntrySchema** -- TypeBox Object with fields:
   - id: String (UUIDv7)
   - timestamp: String (ISO 8601 ms precision)
   - entry_type: LedgerEntryTypeSchema
   - author: EntryAuthorSchema
   - prev_hash: Union(String, Null) -- SHA-256 of previous line
   - signature: String -- Base64-encoded Ed25519 signature over SignableContent
   - encrypted_payload: EncryptedPayloadSchema (import from ./encryption.js)
   - metadata: EntryMetadataSchema
   Export `LedgerEntry = Static<typeof LedgerEntrySchema>`
